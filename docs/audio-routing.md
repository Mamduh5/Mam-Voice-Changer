# Audio routing

Install VB-CABLE, then choose **CABLE Input** as Mam Voice Changer's output. Choose **CABLE Output** as the microphone inside Discord, OBS, TikTok Live Studio, Facebook Live, or a browser.

Use headphones while testing to avoid feedback. Windows sample rates should match when possible (48 kHz is recommended). Do not select speakers as output while an open microphone is nearby.
