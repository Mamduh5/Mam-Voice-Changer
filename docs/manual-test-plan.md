# Manual test plan

1. Confirm all enabled Windows microphones and outputs appear.
2. Start with the default microphone and headphones; verify meters respond and audio is audible.
3. Route to CABLE Input and monitor CABLE Output in OBS.
4. Change each control while live and confirm there is no crash or loud discontinuity.
5. Stop/start ten times and change devices between runs.
6. Run for 30 minutes; record underruns and subjective latency.
7. Verify unplugging a selected device produces a recoverable error on restart.
