# Prototype scope

The first milestone proves the complete local path: physical microphone → Mam Voice Changer → selected Windows output → VB-CABLE → streaming or chat application.

Included: device discovery, live start/stop, low-latency buffering, basic DSP controls, meters, diagnostics, and local-only processing.

Not included: a custom virtual audio driver, recording, cloud processing, AI voice cloning, account systems, or production-grade formant-preserving pitch shifting.
